Wireless Alarm System

Project Overview

In this project I created an alarm system that is controlled over bluetooth from a computer. The alarm system uses an LCD screen to display whether or not it is armed. A passive infrared (PIR) sensor is used to detect motion and sound a passive buzzer if the alarm is in the armed state. To control the state of the system, the computer communicates with a HC-05 bluetooth module connected to the microcontroller. The computer runs a python program which uses the Universal Asynchronous Receiver/Transmitter (UART) protocol to communicate with the bluetooth module.

Specifications

This system uses 1 Nucleo-L476RG development board microcontroller and 1 computer which send commands to each other using the UART protocol over a HC-05 bluetooth module. The microcontroller uses the received commands to control the state of the alarm system. Receiving the message "ALARM_ON" arms the system, "ALARM_OFF" disarms the system, and "SOUND_OFF" plays a tune assuming the buzzer is not playing the alarm. When the system is disarmed, the LCD displays "disarmed" and remains in a dormant state. When the system is armed, the LCD displays "ARMED". When the PIR sensor detects motion, it sends a rising edge interrupt to the microcontroller. If the system is in an armed state, the buzzer will play the alarm sound. Otherwise it will do nothing. This alarm can be placed anywhere. It can be powered by a wire outlet or a battery pack. It can be controlled anywhere within distance of the bluetooth module using a computer with Python installed.

Parts List

  • 1 Nucleo-L476RG microcontroller
  • 1 computer with Python installed
  • 1 1602 LCD screen
  • 1 PIR sensor
  • 1 HC-05 bluetooth module
  • 1 passive buzzer
  • 1 breadboard
  • about 20 jumper wires
